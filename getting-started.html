Manifold is a lightweight reactive state management library for JavaScript/TypeScript. It provides reactive state with automatic dependency tracking and HTML data attribute bindings.

1. Reactive State

3. How It Works

Here's the TypeScript code powering this demo:

import { State, createState } from './src/index';

// Create reactive state
const name = createState('World');
const count = new State(0);

// Computed state (automatically updates)
const greeting = new State(() => `Hello, ${name.value}!`);
const isEven = new State(() => count.value % 2 === 0);

// Effects run when dependencies change
greeting.effect(() => {
    console.log('Greeting changed:', greeting.value);
});

// Update functions
function updateName() { /* cycle through names */ }
function increment() { count.value += 1; }
function decrement() { count.value -= 1; }

4. HTML Data Attributes

Bind HTML elements to reactive state using data attributes:

  • data-bind="textContent: expression" - Update element's text
  • data-if="condition" - Show/hide based on condition
  • data-else - Alternative when condition is false
  • data-sync="variable" - Two-way binding for inputs
  • data-each="array as item" - Repeat elements
