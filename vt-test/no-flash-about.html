Our Story

Notice how smoothly the content changed while the header and navigation stayed perfectly stable? That's the power of properly configured cross-document view transitions!

The key insight is that the View Transitions API applies a default fade to the entire page, but we can override this behavior to only transition specific elements.

What We Did:

  • Disabled root transition: No page-wide fade
  • Persistent elements: Header, nav, footer have stable names
  • Content transitions: Only main content animates
  • Fallback support: Works without View Transitions API

This creates a smooth, app-like experience while maintaining the benefits of traditional multi-page architecture!
