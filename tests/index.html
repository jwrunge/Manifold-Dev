Installation

Copper was made to be flexible, so it can be used all on its own or alongside other libraries and frameworks in a JS build system.

HTML 

            
<!-- Include the script in your HTML - JSDoc provides types for you! -->
<script src="https://cdn/path/to/copper.min.js"></script>

<!-- We recommend downloading the script and serving it yourself -- it's only 10KB! -->
<script src="./scripts/copper.min.js"></script>

<!-- If you need Copper stack traces, you can include the version with source maps. -->
<script src="./scripts/dev.copper.js"></script>
TypeScript 

            
// If you are building or bundling your code already, you can import Copper as a TypeScript module.
import Copper from 'copper';
